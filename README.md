# QOYN Platform

AI-powered educational and professional platform designed to bridge the gap between academic education and labor market requirements.

## Features

- Career recommendation system
- AI-generated projects
- CV analysis
- AI mentor chatbot
- Job readiness simulator
- Project evaluation system
- Company dashboard
- Student dashboard
