# 🌟 About QOYN

QOYN is an advanced AI-powered educational and professional platform designed to bridge the gap between academic education and labor market requirements.

The platform helps students build personalized career pathways through:

* Intelligent skill analysis
* AI-generated projects
* Smart evaluation systems
* Career recommendations
* Job readiness simulations
* AI mentorship systems

QOYN combines multiple Artificial Intelligence paradigms into one integrated ecosystem that improves learning experiences and professional development.

---

# ✨ Why QOYN is Different

Unlike traditional educational platforms, QOYN combines Recommendation Systems, Generative AI, NLP, Computer Vision, Semantic Matching, and AI Evaluation into one integrated intelligent ecosystem.

The platform does not rely on a single chatbot model. Instead, it orchestrates multiple AI components that work together to provide personalized learning, project generation, professional readiness evaluation, and intelligent mentorship.

QOYN acts as an AI orchestration platform where multiple AI paradigms collaborate together inside one workflow.

# 🚀 Core Intelligent Systems

## 🎯 Career Recommendation Engine

Analyzes student skills and interests using ONET and ESCO occupational databases to recommend suitable career paths.

## 🤖 AI Project Generator

Generates personalized project ideas, requirements, and technologies based on the student's target role.

## 📊 AI Project Evaluation

Analyzes uploaded projects using:

* Code analysis
* Runtime analysis
* Screenshot analysis
* Computer Vision
* LLM evaluation

## 🧾 CV & Skill Analysis

Evaluates professional readiness and identifies missing skills.

## 💬 AI Mentor Assistant

Provides intelligent personalized guidance using RAG and LLM systems.

## 🏢 Company Dashboard

Allows companies to review students, projects, and analytics.

## 👨‍🎓 Student Dashboard

Tracks learning progress, submissions, evaluations, and achievements.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yousef-yasin/qoyn-platform.git
```

## Run Frontend

Place the frontend inside the XAMPP `htdocs` folder.

## Run Backend

Start PHP server and FastAPI AI services.

## Database Setup

Import database files from the `database` folder using phpMyAdmin.
