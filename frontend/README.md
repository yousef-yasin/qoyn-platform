# UTBN Web

واجهة الويب (Front-end) للمشروع.

## المحتويات
- صفحات HTML: `index.html`, `login.html`, `signup.html`
- ملفات الستايل: `css/`
- ملفات الجافاسكربت: `js/`

## التشغيل
افتح `index.html` بالمتصفح (أو شغّلها على أي Web Server مثل XAMPP/Apache).

## ملاحظة
ملفات الـ PHP وقاعدة البيانات موجودة في مستودع **utbn-backend**.
