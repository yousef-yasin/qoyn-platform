ضع هنا صورة القالب (المُستخرجة من PDF) للشهادة.

✅ الأفضل: حوّل ملف PDF تبع الشهادة لصورة PNG (جودة عالية) وسميها:
  phase1.png

مثال (على ويندوز): افتح PDF > Print > Microsoft Print to PDF (أو Export) ثم استخدم أي أداة لتحويل PDF إلى PNG.
مثال (على لينكس):
  pdftoppm -png -r 200 your_template.pdf phase1
  ثم خذ ملف phase1-1.png وسمّه phase1.png
